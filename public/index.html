Lumina

Self-hosted service that generates and executes website test cases from video + audio recordings — Whisper transcription, Ollama test generation, Playwright execution.

Features

  • Extract and transcribe Loom-style recordings locally
  • AI-generated structured test cases via Ollama
  • Playwright/Chrome execution with failure screenshots
View on GitHub

Self-hosted setup

python -m venv venv
pip install -r requirements.txt
playwright install chromium
# Requires Ollama at http://localhost:11434

Designed to integrate with Lampshade for spotlight testing.
