Three beats, one pipeline

Capture intent

PMs and designers narrate the happy path while clicking through the product — no selectors or test DSL upfront.

Review every step

Lumina proposes actions from the recording so QA can approve, tweak, or reject before anything hits CI.

Spotlight failures

Failed steps ship screenshots and context so you see exactly which screen regressed — not a vague log line.

Start on your machine

Clone the repo, point Lumina at a walkthrough, and run the full video → Playwright pipeline without sending recordings to a vendor cloud.
