Why teams use Lumina

      

        

          Speak your flow
          PMs and designers record how features should work — no test DSL required upfront.
        

        

          Catch UI regressions
          Failed steps save screenshots so you see exactly what broke, not just a red X.
        

        

          Own your data
          Run the full pipeline on your machines; integrate with Lampshade for spotlight coverage.
        

      

    


    

      
How it works

      

        

          

            Share a walkthrough
            Point Lumina at a screen recording with narration — the kind you already send to stakeholders.
          

        

        

          

            Get executable checks
            Lumina infers clicks, fills, and expectations from what was said and shown on screen.
          

        

        

          

            Ship with confidence
            Re-run after every deploy; JSON reports and failure screenshots keep the team aligned.

Local quick start

      
For the real video → test → Playwright pipeline, install dependencies once:

      
git clone https://github.com/PiyushMishra318/lumina.git
cd lumina && python -m venv venv
pip install -r requirements.txt
python -m playwright install chromium
python lumina/main.py --video your-walkthrough.mp4 --url https://yoursite.com
